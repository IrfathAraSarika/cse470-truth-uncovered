# Backend Service

Node.js + Express backend API for Truth Uncovered.

This service should contain:

- Authentication and role-based authorization
- Citizen, admin, NGO partner, and government officer APIs
- Report, evidence, location, and case workflow APIs
- Duplicate detection and flagged content moderation logic
- Institution ranking, trust score, heatmap, and analytics APIs
- Blog, legal resources, notification, and gamification APIs
